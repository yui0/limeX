GrFreeImage()

Name

GrFreeImage() -- Destroy an image buffer

Synopsis

void GrFreeImage ( GR_IMAGE_ID id );

Description

This function destroys the specified image buffer, and frees the memory that it uses.

Parameters

TypeNameDescription
GR_IMAGE_IDidThe ID of the image buffer.
